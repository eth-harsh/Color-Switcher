Task 1 — Preset buttons

When any color button is clicked, change the .color-box background to that color and update .color-name to show the button's text.
Task 2 — Custom hex input

When .btn-apply is clicked, read the .hex-input value. If it starts with # and has 7 characters → apply it. Otherwise show an error in .message.
Task 3 — Reset button

When .btn-reset is clicked, put everything back to default — box color, color name, input field, message.
Task 4 — Click counter

Every time a color is successfully applied (preset or custom), increment the .count number by 1.
